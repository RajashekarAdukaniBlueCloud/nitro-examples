Variant Examples

Button Tags

The button classes can be applied to <a> and <input> elements as well

Link

When using button classes on <a> elements that are used to trigger in-page functionality instead of linking to pages, be sure to add role="button" so that screen readers can convey their functionality appropiately.

Sizes

Use .btn-lg and .btn-sm classes for button sizing.

Disabled State

Add the disabled attribute to any button element

Disabled Link Buttons

When disabling a link button, use the .disabled class and be sure to add aria-disabled="true" to screen readers.
